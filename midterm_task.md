# CDT517 Mid-term Task

## 任务要求
- **字数:** 2000-2500 words (Word格式)
- **截止日期:** 2026年3月24日
- **提交方式:** Moodle

## 任务内容
分析校园痛点（如"学习效率"），提出解决方案，设计验证方法

## 必须包含
1. Problem Analysis (问题分析)
2. User Research (用户研究)
3. Solution Design (解决方案设计)
4. Verification Method (验证方法)
5. Result Prediction/Adjustment Plans (结果预测/调整计划)

---

## 建议选题

### 选题1: 港硕新生校园适应问题
- **痛点:** 新生对校园资源不熟悉、选课困惑、社交困难
- **解决方案:** 新生导航App + 同伴计划

### 选题2: 跨专业学习效率问题
- **痛点:** 跨专业学生基础知识薄弱、学习进度跟不上
- **解决方案:** 自适应学习系统 + 知识图谱

### 选题3: 小组协作效率问题
- **痛点:** 小组作业沟通成本高、任务分配不均
- **解决方案:** 智能协作平台
